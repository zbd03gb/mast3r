# Introduction:
## MAST3R
**图像匹配**是三维视觉中所有高性能算法与系统的核心问题，但尽管匹配在本质上与相机位姿和场景几何紧密相关、属于一个三维问题，现有方法通常将其作为二维问题来处理。

本文提出**将匹配明确建模为一个三维任务**，基于 DUSt3R 这一 Transformer 驱动的三维重建框架进行扩展，**在点图回归的基础上引入一个新的稠密局部特征输出头，并通过额外的匹配损失进行训练**。

本文在保持 DUSt3R 对极端视角变化高度鲁棒性的同时显著提升了匹配精度，并通过提出一种具有理论保证的快速互易匹配方案，有效缓解了稠密匹配的二次复杂度问题，实现了数量级的加速与性能提升。

## POW3R
三维视觉回归模型在实际应用中往往可以**获得相机内参、相对位姿或深度**等先验信息，但现有前馈式方法在测试时通常无法有效利用这些已知先验。

本文提出了 Pow3R，一种新的大规模三维视觉回归模型，**能够在单一网络中灵活接收并融合图像与任意组合的辅助信息（如内参、相对位姿、稠密或稀疏深度）**。该方法建立在 DUSt3R 的 Transformer 架构之上，通过轻量且通用的条件化机制，将辅助信息作为额外引导，从而在先验可用时预测更精确的结果。

Pow3R 通过在训练阶段随机提供不同模态子集，使模型能够适应测试时不同程度的先验信息，并由此解锁新的能力，如原生分辨率推理和点云补全；实验表明，其在三维重建、深度补全、多视图深度预测、多视图立体和多视图位姿估计等任务上均取得了最先进的性能。

## FAST3R
多视图三维重建在计算机视觉中仍然是核心挑战，尤其是在需要从多样化视角获得准确且可扩展表示的应用中。目前领先的方法如 DUSt3R 主要采用**成对处理**的策略，需要额外的全局对齐步骤来实现多视图重建。

本文提出了 **Fast 3D Reconstruction（Fast3R）**，一种 DUSt3R 的多视图推广方法，通过**一次性处理 N 张图像**实现高效可扩展的三维重建，避免了迭代对齐的开销。Fast3R 基于 Transformer 的架构能够在单次前向传播中并行处理多视图，通过在相机位姿估计和三维重建任务上的大量实验，显示了最先进的性能，同时显著提升了推理速度并减少误差累积。这表明 Fast3R 为多视图应用提供了一个鲁棒且可扩展的替代方案，在提升效率的同时保持了重建精度。

# 项目部署常见问题：
Mast3R:

```python
conda create -n mast3r python=3.10 cmake=3.14.0
conda activate mast3r 
conda install pytorch torchvision pytorch-cuda=11.8 -c pytorch -c nvidia  # use the correct version of cuda for your system
pip install -r requirements.txt
pip install -r dust3r/requirements.txt
# Optional: you can also install additional packages to:
# - add support for HEIC images
# - add required packages for visloc.py
pip install -r dust3r/requirements_optional.txt
```

配置环境时需要安装Mast3R项目的子模块Dust3R的依赖：pip install -r dust3r/requirements.txt

注意在dust3r/requirements.txt中删除“trochvision” 和 “pytroch”

或者使用pip install -r dust3r/requirements.txt --no-deps命令安装依赖

否则会导致pytroch删除重装，导致版本不匹配



Fast3R:

```python
# clone project
git clone https://github.com/facebookresearch/fast3r
cd fast3r

# create conda environment
conda create -n fast3r python=3.10 cmake=3.14.0 -y
conda activate fast3r

# install PyTorch (adjust cuda version according to your system)
conda install pytorch torchvision torchaudio pytorch-cuda=11.8 nvidia/label/cuda-12.4.0::cuda-toolkit -c pytorch -c nvidia

# install requirements
pip install -r requirements.txt

# install fast3r as a package (so you can import fast3r and use it in your own project)
pip install -e .
```

根据官方流程配置环境，无法运行 python fast3r/viz/demo.py ，原因是pip自动下载的gradio版本过新，某些函数被弃用。降级gradio会导致一系列的依赖冲突。

所以需要额外实现脚本调用模型输出点云。



# 输入数据准备
使用nersemble数据集作为3r模型的输入。nersemble数据集提供了16视角的高清人脸采集数据，原始数据以视频的形式保存。本实验选择编号为31的人脸，并截取每个视角下的人脸表情变化视频的第一帧，作为3R模型的输入数据。
